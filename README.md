# AsoulFollower
> Version: 1.0.0
-----------------
## 使用方法
> 在Release下载压缩包

> 解压到文件夹，打开软件

> 使用手机端B站扫码登录（账号信息只保存在本地）

> 进入界面后等待动态刷新

> 列表刷新完成后将开始以10s为一次周期轮询

> 若动态更新，可以选择自动点赞以及气泡通知
